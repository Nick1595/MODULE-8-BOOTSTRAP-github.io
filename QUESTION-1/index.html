QUESTION-1 :- What are the advantages of Bootstrap?

Bootstrap is an open-source CSS framework designed to come up with mobile-friendly, responsive front-end web development. It consists of enormous versatile and reusable pieces of code written in HTML, JavaScript, and CSS. As a framework, fundamentals are already placed for responsive web development, and developers simply need to position the code in a premeditated grid system.

ADVANTAGES OF BOOTSTRAP

  • Easy initiation
  • Responsiveness
  • Highly customizable
  • JavaScript plugins to your rescue
  • Far-reaching popularity
  • Ample documentation
  • Updates at the doorstep
  • Top-notch support
